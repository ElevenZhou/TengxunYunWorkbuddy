# 🤖 AI Agent Games V2

下一代 AI Agent 游戏集，专注评估 LLM 的核心能力。

## 📊 能力评估维度

| 能力 | 游戏 | 评估指标 |
|------|------|----------|
| **推理** | 逻辑谜题、密码破解 | 正确率、推理步数 |
| **规划** | 迷宫逃脱、任务调度 | 完成效率、最优解 |
| **记忆** | 上下文追踪、信息管理 | 多轮准确率 |
| **工具使用** | 命令行冒险、API 调用 | 工具调用正确性 |
| **对话** | 谈判、协作 | 目标达成、策略有效性 |

---

## 🎮 游戏列表

### 1. 密码破译 (Code Breaker)

**能力**：推理 + 模式识别

```
AI 需要破解一个 4 位数字密码
每次猜测后返回：位置正确的数量 + 数字正确但位置错误的数量
挑战：在有限次数内找出密码
```

### 2. 迷宫逃脱 (Maze Escape)

**能力**：规划 + 空间推理

```
AI 在一个 N×N 迷宫中，需要找到出口
每次可以移动：上/下/左/右
挑战：找到最短路径
```

### 3. 二十问 (20 Questions)

**能力**：提问策略 + 信息获取

```
AI 心里想一个物体
人类提问 Yes/No 问题
AI 在 20 个问题内猜到答案
```

### 4. 谈判专家 (Negotiator)

**能力**：对话策略 + 目标达成

```
场景：买卖双方讨价还价
AI 扮演卖家，目标是卖出最高价
人类出价，AI 谈判
```

### 5. 命令行冒险 (CLI Adventure)

**能力**：工具使用 + 系统操作

```
AI 在一个虚拟文件系统中探索
只能使用：ls, cd, cat, grep 等命令
目标：找到特定文件/信息
```

### 6. 记忆大师 (Memory Master)

**能力**：短期记忆 + 信息处理

```
给 AI 一段信息，短暂展示后提问
测试：多轮对话后能否准确回忆细节
```

## 🚀 快速开始

```python
from games import CodeBreaker, MazeGame, CLIAdventure

# 游戏 1: 密码破译
game = CodeBreaker(digits=4, max_attempts=10)
print(game.start())
# AI 猜测: 1234
# 返回: "2 0" (2个位置正确, 0个数字正确但位置错误)

# 游戏 2: 迷宫逃脱
game = MazeGame(size=10)
print(game.get_map())
# AI 移动: "north"
# 返回: 新位置和状态

# 游戏 3: 命令行冒险
game = CLIAdventure()
print(game.get_state())
# AI 执行: "ls -la"
# 返回: 文件列表
```
